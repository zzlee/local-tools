Use the `apply_patch` tool to edit files. It accepts patches in the standard unified diff format.

A unified diff can contain changes for multiple files. Each file's changes start with:
--- original_file
+++ modified_file

Followed by one or more hunks:
@@ -old_start,old_len +new_start,new_len @@
 context line
-removed line
+added line
 context line

To add a new file, use `/dev/null` as the original file:
--- /dev/null
+++ path/to/file
@@ -0,0 +1,3 @@
+line 1
+line 2
+line 3

To delete a file, use `/dev/null` as the modified file:
--- path/to/file
+++ /dev/null
@@ -1,3 +0,0 @@
-line 1
-line 2
-line 3
